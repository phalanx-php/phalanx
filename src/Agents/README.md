# Agents

Supervised agent activity runtime for Phalanx.

Agents is the execution layer above `phalanx-php/ai-providers` provider/transport contracts: activities, turns, effects, grants, hooks, and persistence.

Part of the [Phalanx framework](https://github.com/phalanx-php/phalanx).
