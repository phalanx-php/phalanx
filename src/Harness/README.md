# Phalanx Harness

Starter agent harness app for Phalanx.

Harness composes Theatron UI, Athena execution, Panoply cues, Agora replay, and
Surreal persistence into the user-facing app shell.

Part of the [Phalanx framework](https://github.com/phalanx-php/phalanx).

## Requirements

- PHP `^8.4`
- `ext-openswoole` `^26.2`
- `ext-pcntl`
- `ext-mbstring`

## Run

```bash
php bin/harness
```
