# Phalanx Worker

Worker-process parallelism for Phalanx. Worker offloads CPU-heavy or isolated work to supervised child processes while preserving task-level coordination from the parent runtime.

Part of the [Phalanx framework](https://github.com/phalanx-php/phalanx).
