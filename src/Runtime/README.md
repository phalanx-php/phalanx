# Phalanx Runtime

Managed async runtime for PHP 8.4+ on Swoole. Runtime provides the scope, cancellation, service lifecycle, and concurrency primitives that the rest of Phalanx builds on.

Part of the [Phalanx framework](https://github.com/phalanx-php/phalanx).
