# Phalanx Postgres

Async PostgreSQL integration for Phalanx via `amphp/postgres`, with connection pooling and query/transaction workflows that fit directly into scoped runtime execution.

Part of the [Phalanx monorepo](https://github.com/phalanx-php/phalanx).
