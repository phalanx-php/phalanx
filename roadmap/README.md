# Phalanx Roadmap Material

This folder holds curated source snapshots, specs, and evidence from old POCs and future package ideas. It is not part of the alpha framework surface and is excluded from Composer archives.

## Layout

| Folder | Contents |
| --- | --- |
| `libs/` | Future or deferred first-party libraries: CDP, Postgres, Redis history, Twilio, Eidolon signal client. |
| `tools/` | Future or deferred tools: Dory/PLX/static runtime, RST, Sentinel, bg-agents. |
| `products/` | Product/application experiments: Agent Bridge, agentic UI, ThreePath CLI. |
| `evidence/` | Historical mechanism proofs: Swoole/Aegis substrate work, process claims, old Theatron material. |

## Curation Rules

Only intentional source, specs, docs, configs, and small static assets belong here. Do not add dependency folders, build output, local env files, caches, nested Git repos, PHAR/ZIP artifacts, or machine-specific binaries.
